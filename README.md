苏雨晴推开"拾光"书店的玻璃门时，风铃清脆地响了一声。五月的阳光透过落地窗洒在木地板上，形成一片片菱形的光斑。她深吸一口气，空气中混合着纸张、油墨和咖啡的香气，这是她最喜欢的味道。

"您好，需要点什么？"一个清朗的男声从吧台后传来。

雨晴抬头，看见一个穿着深蓝色围裙的年轻男人正在擦拭咖啡机。他的刘海有些凌乱地搭在额前，笑起来时眼角有细小的纹路。

"一杯热美式，谢谢。"雨晴轻声回答，手指无意识地绞着包带。

"第一次来？"男人问道，手上的动作没停。

雨晴点点头。她确实很少来这家新开的独立书店，要不是为了逃避母亲安排的又一次相亲，她可能永远不会踏进这里。

"推荐你试试我们的招牌拿铁，用的是云南小粒咖啡豆，不会太苦。"男人将一杯清水放在她面前的托盘上，"我是程阳，这家店的咖啡师兼半个老板。"

"苏雨晴。"她简短地自我介绍，犹豫了一下，"那就拿铁吧。"

程阳转身开始制作咖啡，雨晴则走向书架区。作为出版社的文学编辑，她对书籍有着近乎偏执的热爱。指尖划过书脊时，她感到一阵熟悉的安心。

"您的拿铁。"程阳的声音突然在身后响起，雨晴吓了一跳，手中的书差点掉落。

"抱歉，吓到你了。"程阳歉意地笑笑，将咖啡放在旁边的小圆桌上，"《夜航西飞》，很好的选择。"

雨晴有些惊讶："你知道这本书？"

"当然，柏瑞尔·马卡姆的自传。"程阳的眼睛亮了起来，"她是我最敬佩的女性之一，独自飞越大西洋的传奇飞行员。"

雨晴第一次认真打量这个咖啡师。他的眼睛在谈论书籍时闪烁着特别的光彩，与刚才职业化的笑容完全不同。

"很少有咖啡师对文学这么了解。"她忍不住说。

程阳耸耸肩："大学读的中文系，后来发现更喜欢做咖啡。要听听我的故事吗？"他指了指雨晴对面的椅子。

那天下午，雨晴在书店待了整整三个小时。他们从马卡姆聊到博尔赫斯，从咖啡豆的烘焙程度讨论到纸质书与电子书的优劣。雨晴惊讶地发现自己居然对一个刚认识的人说了这么多话——她平时可是连同事聚餐都尽量找借口推掉的。

"你明天还会来吗？"临走时，程阳突然问道，"我们新到了一批二手英文原版书，我想你会喜欢。"

雨晴的心跳漏了一拍："我...我尽量。"

接下来的两周，雨晴几乎每天下班都会去"拾光"。程阳总会为她留一个靠窗的位置，有时会在咖啡杯下压一张写着书摘的便签。雨晴则开始带自己编辑的书稿来看，偶尔会征求程阳的意见。

"你今天心情不好。"一个周五的傍晚，程阳将一杯加了双份焦糖的拿铁放在雨晴面前。

雨晴抬头："这么明显？"

"你翻页的速度比平时快了三分之一，而且已经盯着同一行字看了十分钟。"程阳拉开椅子坐下，"想聊聊吗？"

雨晴叹了口气："我妈妈又给我安排了相亲，明天。"

程阳的手指在桌面上轻轻敲击："不想去？"

"我已经29岁了，在我妈眼里简直就是滞销商品。"雨晴苦笑，"但每次相亲都像在面试，讨论房子、车子、年薪...没人问我喜欢读什么书，或者为什么喜欢雨天。"

程阳沉默了一会儿，突然说："明天书店有周年庆活动，我需要帮手。你可以告诉阿姨你要加班。"

雨晴睁大眼睛："你是说...让我撒谎？"

"我是在给你一个选择。"程阳微笑，"当然，如果你更想去见那位'条件很好'的先生..."

"我选择书店。"雨晴脱口而出，然后为自己的急切感到有些尴尬。

周年庆那天，雨晴帮着程阳布置场地、招待客人。她惊讶地发现，在这个熟悉的环境里，自己居然能自然地与陌生人交谈。当一位老顾客误以为她是老板娘时，程阳只是笑笑没有纠正，雨晴的耳根烧了起来。

活动结束时已是深夜，外面下起了大雨。

"我送你吧。"程阳锁好店门，撑开一把黑色大伞。

雨声敲打着伞面，他们在雨中并肩而行。雨晴闻到了程阳身上淡淡的咖啡香，混合着雨水的清新。

"其实..."程阳突然停下脚步，"我有个秘密要告诉你。"

雨晴的心跳加速："什么？"

"那些'新到的二手书'，有一半是我特意去淘的。"程阳的眼睛在街灯下闪闪发亮，"我只是想找个理由让你多来几次。"

雨晴感到一阵暖流涌过全身。雨滴从伞沿滑落，像一串串水晶帘子将他们与外界隔开。

"我也有个秘密。"她轻声说，"我早就看出来了。"

程阳大笑起来，笑声融入雨声中。他慢慢低下头，雨晴闭上眼睛，感觉到他温热的呼吸...

"要进来避避雨吗？"程阳最终只是在她耳边轻声问道，指了指前方亮着灯的公寓楼。

雨晴点点头，心跳如鼓。她知道，这只是一个开始，就像一本好书的第一章，令人期待又充满可能。
